\# Movie Library (Личная кинотека)



\*\*Автор:\*\* Верещак Арсентий Владимирович



\## Описание проекта

Графическое приложение (GUI), разработанное на языке Python с использованием библиотеки `tkinter`. 

Приложение предназначено для удобного хранения информации о фильмах, ведения личной кинотеки и быстрого поиска по базе. 

Данные сохраняются в локальный файл `movies\_data.json`, что обеспечивает их сохранность между запусками программы.



\## Основной функционал

1\. \*\*Добавление фильмов:\*\* Ввод названия, жанра, года выпуска и личного рейтинга.

2\. \*\*Валидация данных:\*\*

&#x20;  - Год выпуска проверяется на принадлежность к целочисленному типу (число).

&#x20;  - Рейтинг строго ограничен диапазоном от 0 до 10.

3\. \*\*Хранение данных:\*\* Автоматическая сериализация и десериализация базы фильмов через JSON.

4\. \*\*Фильтрация и поиск:\*\* 

&#x20;  - Поиск фильмов по конкретному жанру.

&#x20;  - Поиск фильмов по году выпуска.

&#x20;  - Возможность сброса фильтров для отображения всего списка.
